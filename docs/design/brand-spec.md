## Layout Posture

- White enterprise canvas with cool blue wash only in the hero, not across the whole page.
- Strong centered solution card grid with thin borders, modest shadows, and large line icons.
- One clear navy/blue CTA pair in the hero and one deep navy conversion band near the bottom.
- Security visualization should feel like a composed product diagram, not a pasted stock image.
- Footer returns to solid deep navy to anchor the corporate tone.
